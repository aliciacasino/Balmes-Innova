Sobre el curso

¿Te apasiona el mundo digital y la creación de páginas web? ¡Éste es tu curso! El objetivo de esta enseñanza es crear y publicar páginas web que integren textos, imágenes y otros elementos a través de lenguajes de marcas. También, realizarás los procedimientos de instalación y verificación en el servidor correspondiente.

Además, concluirás con unas prácticas que te permitirán aplicar los conocimientos teóricos expuestos, adquiriendo las habilidades necesarias para el correcto desarrollo del puesto de trabajo.

El perfil profesional con el que saldrás en este curso es el de Front-End Developer.

Contenido

HTML

Es el lenguaje más básico que se utiliza para crear las páginas webs. Utiliza etiquetas rodeadas por “<” y “>” que contienen la información sobre el significado y la estructura del contenido de la web.

Es muy importante tener bien organizado este lenguaje, pues será la base para desarrollar la página web.

CSS

Es el lenguaje que da estilo y aspecto a la página creada por HTML. Se puede modificar la fuente, los colores, los tamaños y otras características con el objetivo de crear el diseño más bonito.

Tienes el poder de diseñar la página de manera visual, a tu gusto, dándole un toque más personal.

JavaScript

Es un lenguaje de guiones del lado del cliente basado en objetos para poder crear efectos especiales en las páginas para así definir una relación más interactiva con el usuario.

Imagina crear efectos en los textos, animaciones en las imágenes, validar formularios… En general, todo aquello que enganchará al usuario a tu web.

WordPress

Es un sistema de gestión de contenidos que facilita la creación de webs de manera sencilla y rápida. Un tercio de todos los sitios webs del mundo están hechos en WordPress.

Una vez aprendido los lenguajes anteriores sabrás cómo funciona Wordpress y así podrás personalizar mucho más las plantillas y marcar la diferencia.
